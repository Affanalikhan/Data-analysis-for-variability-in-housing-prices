## 🎯 Overview

This project investigates the factors that contribute to price variability in the housing market. Using statistical techniques and machine learning, it identifies which property features have the most influence on price fluctuations and builds models to quantify that relationship.

---

## 🔍 Analysis Breakdown

### 1. Exploratory Data Analysis
- Price distribution and skewness analysis
- Correlation analysis between features and price
- Geographic and categorical breakdowns
- Outlier identification

### 2. Variability Analysis
- Variance and standard deviation across feature groups
- ANOVA tests for categorical variables
- Coefficient of variation analysis
- Price range segmentation

### 3. Feature Engineering
- Handling missing values
- Encoding categorical features
- Log transformation for skewed distributions
- Feature selection based on importance scores

### 4. Predictive Modeling
- Multiple Linear Regression
- Ridge and Lasso Regression
- Gradient Boosting (XGBoost / sklearn)
- Cross-validation and hyperparameter tuning

### 5. Key Insights
- Location and neighborhood quality are the dominant price drivers
- Square footage has a near-linear relationship with price up to a threshold
- Age of property interacts with renovation status significantly
- Seasonal patterns contribute to short-term price variability
